# Quick setup (basic, local and personnal)
## MongoDB (part 1/4)
Start the database.

```
cd /Applications/NicolasPrograms/database/MongoDB/mongodb-osx-x86_64-3.2.6
bin/mongod
```

The datas are in stored in ***/data/db***.

Usefull GUI to see the datas: **MondoDB Compass**.
